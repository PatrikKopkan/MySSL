Nainstalujte balíčky z dependencies.txt: pip install "balíček"
pro spravné fungování kodu se pohybujte v složce projektu (cd MySSL)

pro vytvoření vlastních testovacích certifikátů spusťte certificates.py a změnte cestu v config1 nebo config2

pro test komunikace spusťte server.py a client.py

komunikace bude fungovat jako u vysílačky, server pošle zprávu a čeká na odpoveď od klienta vice versa.
